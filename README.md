## Setup
* `npm install`
* `npm install webpack -g`
* `npm install karma -g`

## Development
* `npm run watch`
* `npm run unit` (Does not currently work)

## Minify/uglify/prodify
* `npm run build`

## Includes configs for:
* Webpack
* ESLint
* Babel
* Karma
* React hot swapping

## Supports
* SCSS/Bourbon
* React
* Redux

## Caveats
* React hot swapping only works with class-based components; not stateless components.
